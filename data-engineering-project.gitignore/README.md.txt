# 📊 Marketing Data Engineering Project

## 📌 Overview
This project builds an end-to-end data pipeline using Databricks and visualizes insights using Power BI.

## 🛠 Tools Used
- Databricks
- PySpark
- SQL
- Power BI

## 🔄 Workflow
1. Data Ingestion
2. Data Cleaning
3. Data Transformation
4. Output Generation
5. Dashboard Visualization

## 📊 KPIs
- CTR (Click Through Rate)
- Conversion Rate
- ROI

## 📁 Project Structure
- notebooks/ → ETL code
- data/ → input data
- outputs/ → processed data
- powerbi/ → dashboard files
